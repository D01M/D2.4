Features:
• Multi-source real seismic data
• USGS 6-month global backbone
• EMSC/CSEM global feed
• GeoNet New Zealand feed
• INGV Italy feed where browser access allows
• GFZ/GEOFON feed where browser access allows
• Active-only seismic station layer from FDSN station services
• Static Georgian station fallback from IliaUni public station list
• Station waveform/audio preview and source links
• Station sample-rate Hz display where channel metadata exposes it
• Nearby real-catalog station activity meter
• Magnitude filtering
• Offline cache support

Credit: Georgian station metadata references publicly available information from Ilia State University / Institute of Earth Sciences and National Seismic Monitoring Center.

Note: Not affiliated with or endorsed by Ilia State University. Some agencies may block browser requests with CORS. The app will keep working with sources that respond.
